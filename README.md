## js_sizers
> Show your javascript file size after minification and gzip

### Install

```sh
cargo install js_sizers
```

### Usage

```sh
js_sizers ./test.js
```

```sh
# Output
Reading file: ./test.js...

origin: 43235 bytes
minified: 10091 bytes
gzipped: 4167 bytes
```
